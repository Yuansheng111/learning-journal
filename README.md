# Learning Journal 學習日誌

Self-study notes for building mathematical intuition and ML foundations.  
系統性自學數學直覺與機器學習基礎的筆記。

This repository is an [Obsidian](https://obsidian.md/) vault — all notes are written in Markdown with `[[wikilinks]]` for bidirectional linking, and best viewed by opening the repo folder directly in Obsidian.  
本 repo 同時是一個 Obsidian vault，使用 `[[雙向連結]]` 串聯概念，建議用 Obsidian 開啟此資料夾以獲得最佳閱讀體驗。

## What's Inside 內容說明

### 3Blue1Brown Notes 筆記

Bilingual (中文 / English) notes for three 3B1B series, studied in this order:  
三個 3B1B 系列的中英對照筆記，按以下順序學習：

1. **Neural Networks** — Build ML intuition first  
   **神經網路** — 先建立 ML 直覺
2. **Essence of Linear Algebra** — Spatial & geometric intuition for matrices  
   **線性代數的本質** — 矩陣的空間與幾何直覺
3. **Essence of Calculus** — Chain rule = the math behind backpropagation  
   **微積分的本質** — 鏈式法則 = 反向傳播的數學本體

Each episode note follows a consistent template: one-line summary, key intuitions, key terms (bilingual glossary), personal relevance, and honest "still unclear" items for review.  
每集筆記使用統一模板：一句話總結、關鍵直覺、重要術語（中英對照）、跟我的關聯、以及誠實記錄「還沒搞懂的」作為複習重點。

### Concept Library 概念庫

Standalone notes for core concepts that appear across multiple episodes. These form a growing knowledge graph visible in Obsidian's Graph View.  
獨立的核心概念筆記，串聯跨系列中重複出現的概念。隨著學習累積，會在 Obsidian 的 Graph View 中形成知識圖譜。

## How to Use 使用方式

1. Clone this repo  
   複製此 repo
2. Open the root folder in [Obsidian](https://obsidian.md/)  
   用 Obsidian 開啟根目錄
3. Navigate from `3B1B/HOME.md` as the entry point  
   從 `3B1B/HOME.md` 開始瀏覽

> **Note:** `[[wikilinks]]` are an Obsidian feature and will not render as clickable links on GitHub. The content is still fully readable as plain Markdown.  
> **備註：** `[[雙向連結]]` 是 Obsidian 的功能，在 GitHub 上不會顯示為可點擊的連結，但內容仍可作為純 Markdown 閱讀。
