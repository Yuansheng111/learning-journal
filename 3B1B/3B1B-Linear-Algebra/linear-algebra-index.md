# 3B1B - Essence of Linear Algebra 線性代數的本質

## 系列概覽 Series Overview

用幾何直覺理解線性代數，不再把矩陣當成數字表格。ML 的數學基底——理解了線性轉換，PCA、SVD、神經網路的權重矩陣都會通。
Geometric intuition for linear algebra. The mathematical foundation for ML — once you understand linear transformations, PCA, SVD, and weight matrices all click.

## 重點章節 Priority Chapters

- **必看 Must watch**：Ch 1-7（向量 vectors、線性組合 linear combinations、線性轉換 linear transformations、行列式 determinant、逆矩陣 inverse matrices）
- **必看 Must watch**：Ch 9-10（點積 dot product、叉積 cross product）
- **重中之重 Critical**：Ch 13-15（特徵向量 eigenvectors、特徵值 eigenvalues、抽象向量空間 abstract vector spaces）
- 可快看 Optional：Ch 8, 11-12
